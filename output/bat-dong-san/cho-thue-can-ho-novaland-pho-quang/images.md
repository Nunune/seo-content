## Hướng dẫn convert sang WebP

**Online:** https://squoosh.app → kéo ảnh → chọn WebP → Quality 85 → Download

**CLI:**
```
cwebp input.png -o output.webp -q 85
```

**Cài libwebp trên Windows:**
```powershell
winget install libwebp
```
