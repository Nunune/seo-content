Đặt file Google Service Account JSON vào đây với tên: service_account.json

Cách tạo:
1. Vào https://console.cloud.google.com
2. Tạo project mới (hoặc chọn project có sẵn)
3. Bật Google Sheets API và Google Drive API
4. Tạo Service Account -> Tạo key -> Chọn JSON -> Tải về
5. Đổi tên file thành service_account.json và đặt vào thư mục này
6. Mở Google Sheet muốn ghi -> Chia sẻ cho email của service account (dạng xxx@xxx.iam.gserviceaccount.com)
