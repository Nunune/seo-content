## Hướng dẫn chụp screenshot TikTok

1. Mở TikTok app trên điện thoại
2. Chụp màn hình ở kích thước tối thiểu 1080x1920px
3. Đặt file vào thư mục `images/` với đúng tên file
4. Chạy lại export: `python _scripts/convert.py _preprocessed.md export-vi-pham-chinh-sach-tiktok.docx`
